# Pytorch

### Commonly Used
- [TORCH.ZEROS](https://pytorch.org/docs/stable/generated/torch.zeros.html): Returns a tensor filled with the scalar value 0, with the shape defined by the variable argument `size`.

###### Example
```
>>> torch.zeros(2, 3)
tensor([[ 0.,  0.,  0.],
        [ 0.,  0.,  0.]])

>>> torch.zeros(5)
tensor([ 0.,  0.,  0.,  0.,  0.])
```
